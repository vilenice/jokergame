## 本地调试
如下本地调试，请根据你的运行环境安装 [Node.js](https://nodejs.org)，并进行如下操作：
```
# 克隆本仓库
git clone https://github.com/vilenice/jokergame.git

# 进入仓库根目录
cd jokergame

# 安装依赖
npm install
```
执行完上述操作，即可进行本地调试：
```
npm run docs:dev
```
